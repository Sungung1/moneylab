# Moneylab

## Overview

Moneylab is a web project for financial calculation workflows and interactive user-facing analysis tools built with the Next.js App Router.

## Tech Stack

- TypeScript
- Next.js
- React
- CSS

## Architecture

Input form -> Calculation engine -> Formatted result output

## Usage

```bash
npm install
npm run dev
```

## Results

The application delivers multiple browser-based finance calculators with reusable business logic, clean UI composition, and a production-style Next.js project layout.
